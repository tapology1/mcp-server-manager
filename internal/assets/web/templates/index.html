How It Works

MCP Server Manager centralizes control of Model Context Protocol servers across different AI clients and target config files.

Controls

  • Client-specific toggles: Enable or disable servers per target
  • Sync All Clients: Apply the current config to each target file
  • Restart Manager: Reload the YAML config without using the terminal

Workflow

  1. Edit your YAML config
  2. Click Restart Manager
  3. Click Sync All Clients
  4. Restart AI clients if they cache MCP state
